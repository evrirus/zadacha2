# Инструмент визуализации графа зависимостей для менеджера пакетов

## Возможности

### Этап 1
- Чтение настроек из CSV-конфигурации.
- Валидация параметров.
- Вывод параметров в формате `ключ = значение`.
- Обработка ошибок.

### Этап 2
- Загрузка `APKINDEX.tar.gz` из Alpine-репозитория.
- Получение прямых зависимостей выбранного пакета.
- Вывод списка зависимостей.

### Этап 3
- Построение полного графа зависимостей (транзитивность).
- Итеративный DFS (без рекурсии).
- Обработка циклических зависимостей.
- Фильтрация пакетов по подстроке.
- Режим тестирования через локальный файл графа.
- ASCII-дерево зависимостей.

---
## Скачка
```
git clone https://github.com/evrirus/zadacha2.git
cd zadacha2
```
## Запуск
```
python main.py config.csv
```
Удаленный файл
```
busybox,https://dl-cdn.alpinelinux.org/alpine/v3.18/main/x86_64/,remote,on,busy
```

Параметры (первая строка конфига)
```
package_name,repo_url_or_path,repo_mode,ascii_tree,filter_substring
```
Локальный файл
```
busybox,C:\Users\kovalskiy\Downloads\APKINDEX.tar.gz,local,on,busy
```
